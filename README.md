# Red Guardian AI
### AI-powered moderation support for Reddit communities
Red Guardian AI is a moderation assistant built for Reddit moderators to make handling reports, spam, harassment, and toxic discussions faster and easier.
The idea behind this project was simple:
Moderators often deal with hundreds of reports and long toxic threads manually. Red Guardian AI helps reduce that workload by using AI to summarize situations, identify risky behavior, and suggest moderation actions in a clean interactive dashboard.
Built during the Reddit Mod Tools Hackathon using Devvit, React, TypeScript, and OpenRouter AI.
---
# The Problem
Managing large online communities is difficult.
Moderators constantly deal with:
* spam attacks
* harassment
* toxic discussions
* hate speech
* repeated reports
* risky users
Reviewing everything manually takes time and can easily become overwhelming.
---
# What Red Guardian AI Does
Red Guardian AI works like an AI co-moderator.
It helps moderators by:
* highlighting risky users
* generating AI summaries of toxic threads
* prioritizing important reports
* suggesting moderation actions
* simulating live moderation monitoring
The goal is not to replace moderators, but to help them respond faster and more effectively.
